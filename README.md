# wx-robot

#### 介绍
基于wechaty，个人微信机器人。群聊，私聊，一个简单的demo。


#### 软件架构
wechaty: https://wechaty.js.org/


#### 安装教程

建议node版本>16

1.  npm install 
2.  npm run dev

运行起来之后就可以扫码登录了。

#### 使用说明

这只是个最基础的框架，只实现了群聊回复功能。

更多功能可以参考: https://wechaty.gitbook.io/wechaty/v/zh/example#basic

申请天行API: https://www.tianapi.com/
